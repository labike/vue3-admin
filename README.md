# vue3-admin
> vue3管理后台, 包含国际化, 动态菜单，权限处理, 换肤, 列表上传下载等, 不包含动态添加菜单

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
